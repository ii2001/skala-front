# SKALA Front

SKALA Full-Stack Engineering 과정에서 진행하는 프런트엔드 과제 저장소입니다.
현재는 JavaScript를 적용하기 전 단계로, 시맨틱 HTML과 순수 CSS를 이용해 여러 주제의 반응형 페이지를 구현했습니다.

## 현재 구현 상태

- 7개의 HTML 페이지와 공통 내비게이션
- Flexbox와 Grid를 활용한 반응형 레이아웃
- 강의 시간표, 회원가입 폼, 로컬 여행 미디어
- 공통 스타일, transition, 제한적인 animation, reduced-motion 대응
- Noto Sans KR Google Fonts 공식 연결
- JavaScript 파일 및 `script` 태그 미사용

## 페이지

- `html/index.html`: 전체 페이지 포털
- `html/holiday.html`: 휴일 일과
- `html/myProfile.html`: 자기소개
- `html/myClass.html`: 주간 강의 시간표
- `html/myTrip.html`: 제주도 여행과 로컬 미디어
- `html/signUp.html`: HTML 회원가입 폼
- `html/signUpResult.html`: 정적 회원가입 완료 화면

## 실행 방법

빌드나 패키지 설치 없이 `html/index.html`을 브라우저 또는 VS Code Live Server로 열어 확인할 수 있습니다.

## 미디어 출처 및 라이선스

여행 페이지는 `media/` 폴더의 로컬 이미지 3개, 오디오 1개, 동영상 1개를 사용합니다.
현재 저장소와 Git 기록만으로는 이 파일들의 원출처와 라이선스를 확인할 수 없습니다.
제출 전 파일 소유자에게 사용 권한을 확인하고 정확한 출처 및 라이선스 정보를 이 문서에 추가해야 합니다.
