기본 소개

현재 SKALA에서 풀스택 개발을 학습하며 개발 역량을 차근차근 쌓고 있습니다.

프로그래밍을 통해 새로운 것을 만들고, 논리적으로 문제를 해결하는 과정에 관심이 많습니다.

내가 좋아하는 음식

  • 소고기와 스테이크: 풍부한 육즙과 고소한 맛을 좋아합니다.
  • 매콤한 떡볶이: 쫄깃한 식감과 기분 좋은 매운맛을 즐깁니다.
  • 초밥과 회: 신선한 재료 본연의 담백한 맛이 매력적입니다.
  • 치킨: 바삭한 식감과 다양한 양념을 골라 먹는 재미가 있습니다.
  • 커피와 디저트: 달콤한 디저트와 향긋한 커피를 함께 즐기면 여유를 느낄 수 있습니다.

올해 할 일

  1. SKALA 교육과정 성실하게 수료하기
  2. HTML, CSS, JavaScript 기초 완성하기
  3. 개인 웹 프로젝트 완성하기
  4. 협업 프로젝트에서 Git과 GitHub 활용하기
  5. 개발 포트폴리오 정리하기

개발자로서의 목표

사용자 경험을 고려하고 사용자가 겪는 문제에 실질적인 해결책을 만드는 개발자가 되고 싶습니다.

단순히 코드를 작성하는 데 그치지 않고 동료와 원활하게 협업하며, 읽기 쉽고 유지보수하기 좋은 결과물을 만드는 것을 목표로 합니다.

좋은 개발은 기능 구현에서 끝나지 않고 사용자의 문제 해결로 완성됩니다.
