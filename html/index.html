현재 학습 목표

시맨틱 HTML로 각 페이지의 의미와 구조를 분명하게 구성하고 있습니다.

공통 스타일과 반응형 레이아웃을 적용했으며, 이제 JavaScript 기초 기능을 실습하고 있습니다.

JavaScript 실습

버튼을 눌러 숫자 맞추기, 성적 계산, 가방 목록 출력 기능을 직접 실행해 보세요.

Up-Down 숫자 맞추기

1부터 50 사이의 숫자를 입력하고 Up과 Down 안내에 따라 정답을 맞혀 보세요.

게임을 시작하면 결과가 여기에 표시됩니다.

성적 계산기

HTML, CSS, JavaScript 점수를 입력해 총점, 평균, 등급과 합격 여부를 확인하세요.

점수 입력을 마치면 계산 결과가 여기에 표시됩니다.

내 가방 보기

SKALA 개발 교육에 챙겨 다니는 소지품과 전체 수량을 목록으로 확인하세요.

버튼을 누르면 가방 속 물품 목록이 여기에 표시됩니다.
